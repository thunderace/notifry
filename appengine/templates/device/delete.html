Delete device

You're about to delete the device ${device.deviceNickname|h}.

Some things you should know:

  • The device will be asked to deregister itself. If you want to use it again, you'll need to make sure you register it again.
  • There is a small chance that the device won't know it's been deregistered. Please keep this in mind.
  • This will stop any notifications being sent to the device.
  • If the device doesn't physically exist or have the software installed, this will only remove the device from your list.
  • This operation can not be undone.
